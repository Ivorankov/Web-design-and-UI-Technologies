This program calculates distances between points(and other stuff)

For your convenience the points are preset:

    p1: (-2, 1)
    p2: (1, 5)
    p3: (5, 5)
    p4: (1, 1)
    p5: (-5, 3)
    p6: (0, 1)

Test will calculate the length of the 3 lines formed by(p1,p2), (p3,p4), (p5,p6).
It will also check if the 3 lines can form a triangle.
